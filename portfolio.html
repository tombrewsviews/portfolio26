debug-toolkit

// npm_package · mcp_server · typescript · 3451_loc

1. AI agents are great at writing code. They're terrible at seeing it run.

2. When an agent hits a bug, it reads code and guesses a fix. You run it, paste the error back, the agent guesses again. Repeat 5–8 times. I built the exit ramp: an MCP server giving agents a closed-loop debugging system. Investigate → instrument → capture → verify → remember.

3. 15 files · 3,451 lines of TypeScript · 4 npm dependencies · JS/TS, Python, Go, Rust/Tauri.
